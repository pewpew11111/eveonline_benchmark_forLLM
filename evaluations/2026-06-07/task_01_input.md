# 이브온라인(EVE Online) 웜홀 종류

이브온라인의 웜홀은 크게 **클래스(CLASS)**별로 구분됩니다.

---

## 📊 클래스별 분류

| 클래스 | 이름 (스타 색상) | 특징 |
|--------|----------------|------|
| **C1** | 화이트 스타 (White Star) | 소규모, 비교적 안전 |
| **C2** | 블루 스타 (Blue Star) | 중규모, 중간 난이도 |
| **C3** | 레드 스타 (Red Star) | 중간 규모 |
| **C4** | 블랙 스타 (Black Star) | 대규모 PvP 가능 |
| **C5** | 마젠타 스타 (Magenta Star) | 대규모, 고가 자원 |
| **C6** | 레드 스타 (Red Star) | 대규모, 극한의 위험 |

---

## 🔗 기타 특수 웜홀

### **Thera (테라)**
- PvP/PvE를 위해 만들어진 **전투형 웜홀**
- 주요 허브(제이드湾, 우카카 등)와 직접 연결
- 일반 웜홀과 달리 항상 존재하는 **고정형**

### **Shattered Wormholes (파편화 웜홀)**
- C5/C6에 **극히 드물게** 생성
- 파편화된 행성이 있어 **이스크anya 값이 매우 높음**
- 위험도도 극도로 높음

---

## ⚙️ 주요 속성 비교

각 웜홀은 아래 속성들이 서로 다릅니다:

- **수명 (Lifetime)** — 얼마나开着 유지되는지
- **질량 한도 (Mass)** — 통과 가능한 선박 총 질량
- **항성 유형** — 각 클래스별 고유한 태양
- **생물 군계 (Wormhole Effects)** — PvE 환경 (방어막, 속도 등)
- **연결 가능 시스템** — 어떤 클래스와 연결되는지

## 🔢 클래스 기준 (가장 일반적인 분류)

웜홀은 **6가지 클래스(W1~W6)**로 나뉩니다:

| 클래스 | 허용 함선 크기 | 특징 |
|--------|-------------|------|
| **W1** | 프리깃만 | 가장 작고 안전, 초보자용 |
| **W2** | 프리깃/구축함 | 소규모 PvP 가능 |
| **W3** | 크루저까지 | 가장 흔한 클래스, 좋은 수익 |
| **W4** | 배틀크루저까지 | 중규모 전투 가능 |
| **W5** | 전함까지 | 대규모 PvP, 고수익 |
| **W6** | 캐피탈까지 | 가장 위험, 최고 수익 |

## 🔗 연결 유형 기준

웜홀이 연결되는 곳에 따라서도 분류됩니다:

- **하이섹 연결** (High-sec) - 안전지대 연결
- **로우섹 연결** (Low-sec) - 로우섹 연결
- **널섹 연결** (Null-sec) - 널섹 연결 (가장 수익 높음)
- **W-space 연결** - 다른 웜홀 연결 (체인 가능)

## ⭐ 특수 웜홀

- **테라(Thera) 웜홀** - 테라 시스템으로 연결
- **에테리움 리치(Etherium Reach)** 웜홀
